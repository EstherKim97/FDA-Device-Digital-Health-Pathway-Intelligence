# Regulatory Pathway Test Cases

This document contains example product scenarios used to test the **FDA Device / Digital Health Pathway Intelligence** dashboard.

These cases help evaluate whether the scoring model produces reasonable regulatory pathway recommendations.

---

# 1. ECG Monitoring Software

**Keyword**  
ECG monitor

**Intended Use**  
Software that analyzes ECG signals and displays rhythm information to help clinicians identify possible arrhythmia patterns.

**Regulatory Framing**

- Supports diagnosis: Yes  
- Supports treatment decisions: No  
- Patient-specific output: Yes  
- Clinician can independently review the basis: Yes  
- Intended for healthcare professionals: Yes  
- Risk level: Moderate  
- Predicate known: Yes  
- Novelty: Low  
- Serious condition: No  
- Unmet need: No  

**Expected Pathway**  
510(k)
